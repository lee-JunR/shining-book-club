# 커튼 사이즈 계산기

가로/세로 사이즈를 입력하면 커튼 겉지 폭수, 커튼 속지 마수, 블라인드 헤베를 계산하는 React 서비스입니다.

커튼 계산기와 블라인드 계산기를 탭으로 분리했습니다. 블라인드는 창 구간 수를 입력하면 구간별 가로, 세로, 설치 개수, 바깥쪽 5cm 여유 적용 여부를 각각 입력할 수 있습니다.

## 실행

```sh
npm install
npm run dev
```

## 기본 계산 기준

자세한 계산법은 [CALCULATION.md](/Users/junr/Dev/2026/project/curtain_size_calculator/CALCULATION.md)에 정리했습니다.

- 커튼(겉지): `ceil(가로(cm) * 주름 배수 / 140cm)`
- 민자주름: 1.5배
- 나비주름: 2.0배
- 커튼(속지): `ceil(가로(cm) * 2 / 90cm)`
- 블라인드 여유 체크된 구간: `(실측 가로 + 5cm) / 100`
- 블라인드 여유 체크 안 된 구간: `실측 가로 / 100`
- 블라인드 높이: `(실측 세로 + 10cm) / 100`
- 블라인드 개별 헤베: `ceil(기본 면적 * 2) / 2`
- 개별 헤베가 1.5㎡ 이하이면 2.0㎡로 고정
- 창별 최종 헤베: `개별 헤베 × 해당 창 개수`
- 총 헤베: `창별 최종 헤베 합산`

NAS 배포용 단일 파일은 `npm run build:nas`로 생성합니다.
