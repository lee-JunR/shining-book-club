# 계산법 문서

이 문서는 현재 계산기에 적용된 커튼, 블라인드 계산 기준을 정리한 문서입니다.

## 공통 원칙

- 사용자가 입력하는 길이 단위는 모두 `cm`입니다.
- 계산 중 면적을 구할 때는 `cm`를 `m`로 변환합니다.
- 원단이나 헤베가 부족하지 않도록 소수점이 생기면 정해진 기준으로 올림 처리합니다.

## 커튼 계산

### 입력값

- 가로 실측치 `W`
- 세로 실측치 `H`
- 나비주름 사용 여부

현재 커튼 결과 계산에는 `W`가 사용됩니다. `H`는 실측 안내용 입력값으로 유지합니다.

### 기준값

- 겉지 1폭: `140cm`
- 속지 1마: `90cm`
- 민자주름 배수: `1.5`
- 나비주름 배수: `2.0`
- 속지 배수: `2.0`

### 커튼 겉지 폭수

나비주름을 끄면 민자주름 `1.5배`, 켜면 나비주름 `2.0배`를 적용합니다.

```text
겉지 계산값 = W × 주름배수 ÷ 140
겉지 폭수 = ceil(겉지 계산값)
```

예시:

```text
W = 300cm, 민자주름
300 × 1.5 ÷ 140 = 3.214...
겉지 폭수 = 4폭
```

```text
W = 300cm, 나비주름
300 × 2.0 ÷ 140 = 4.285...
겉지 폭수 = 5폭
```

### 커튼 속지 마수

속지는 항상 `2.0배`로 계산합니다.

```text
속지 계산값 = W × 2 ÷ 90
속지 마수 = ceil(속지 계산값)
```

예시:

```text
W = 300cm
300 × 2 ÷ 90 = 6.666...
속지 마수 = 7마
```

## 블라인드 계산

블라인드는 커튼 계산과 독립적으로 계산합니다. 창 구간 수를 입력하고, 각 구간별 가로, 세로, 설치 개수를 입력합니다.

### 입력값

각 창 구간마다 다음 값을 입력합니다.

- 구간 가로 실측치 `W`
- 구간 세로 실측치 `H`
- 설치 개수 `Q`
- 양끝 바깥쪽 5cm 여유 적용 여부

세로값은 한 구간에 입력하면 모든 구간에 같은 값으로 동기화됩니다.

### 기준값

- 바깥쪽 가로 여유: `5cm`
- 하단 세로 여유: `10cm`
- 헤베 올림 단위: `0.5헤베`
- 최소 헤베 기준: `1.5헤베 이하`
- 최소 적용 헤베: `2.0헤베`

### 가로 여유 적용

가로 여유 체크박스는 양끝 구간에만 표시합니다.

- 창 구간이 1개인 경우: 체크 시 좌우에 각각 `5cm`씩, 총 `10cm`를 더합니다.
- 창 구간이 2개 이상인 경우: 첫 구간과 마지막 구간만 체크할 수 있고, 체크 시 해당 구간에 `5cm`를 더합니다.
- 가운데 구간은 여유분을 더하지 않습니다.

```text
적용 가로(m) = (W + 가로 여유 cm) ÷ 100
```

### 세로 여유 적용

세로는 하단 여유 `10cm`를 더해 계산합니다.

화면 표기 예시:

```text
260+10cm(하단 10cm여유)
```

계산식:

```text
적용 세로(m) = (H + 10) ÷ 100
```

### 개별 헤베 계산

먼저 기본 면적을 구합니다.

```text
기본 면적 = 적용 가로(m) × 적용 세로(m)
```

그 다음 `0.5헤베` 단위로 올림합니다.

```text
0.5 단위 올림 헤베 = ceil(기본 면적 ÷ 0.5) × 0.5
```

코드에서는 같은 의미로 아래처럼 계산합니다.

```text
0.5 단위 올림 헤베 = ceil(기본 면적 × 2) ÷ 2
```

### 최소 헤베 적용

0.5 단위로 올림한 값이 `1.5헤베 이하`이면 최종 헤베를 `2.0헤베`로 고정합니다.

```text
최종 개별 헤베 =
  0.5 단위 올림 헤베 <= 1.5 이면 2.0
  아니면 0.5 단위 올림 헤베
```

### 설치 개수 반영

각 구간의 최종 헤베에 설치 개수를 곱합니다.

```text
구간 총 헤베 = 최종 개별 헤베 × Q
```

### 전체 블라인드 헤베

모든 창 구간의 총 헤베를 더합니다.

```text
전체 헤베 = 모든 구간 총 헤베 합산
```

## 블라인드 계산 예시

### 예시 1: 3구간 기본값

조건:

- 창 구간 수: 3개
- 각 구간 가로: `100cm`
- 세로: `230cm`
- 설치 개수: 각 `1개`
- 1번, 3번 구간 바깥쪽 여유 체크

1번 구간:

```text
적용 가로 = (100 + 5) ÷ 100 = 1.05m
적용 세로 = (230 + 10) ÷ 100 = 2.4m
기본 면적 = 1.05 × 2.4 = 2.52㎡
0.5 단위 올림 = 3.0헤베
구간 총 헤베 = 3.0 × 1 = 3.0헤베
```

2번 구간:

```text
적용 가로 = 100 ÷ 100 = 1.0m
적용 세로 = (230 + 10) ÷ 100 = 2.4m
기본 면적 = 1.0 × 2.4 = 2.4㎡
0.5 단위 올림 = 2.5헤베
구간 총 헤베 = 2.5 × 1 = 2.5헤베
```

3번 구간은 1번 구간과 같으므로 `3.0헤베`입니다.

```text
전체 헤베 = 3.0 + 2.5 + 3.0 = 8.5헤베
```

### 예시 2: 창 구간이 1개인 경우

조건:

- 가로: `100cm`
- 세로: `230cm`
- 설치 개수: `1개`
- 바깥쪽 여유 체크

창 구간이 1개이면 좌우에 각각 `5cm`씩 더해 총 `10cm`를 더합니다.

```text
적용 가로 = (100 + 10) ÷ 100 = 1.1m
적용 세로 = (230 + 10) ÷ 100 = 2.4m
기본 면적 = 1.1 × 2.4 = 2.64㎡
0.5 단위 올림 = 3.0헤베
전체 헤베 = 3.0헤베
```

## 화면 표시 규칙

- 블라인드 창 그림의 구간 너비는 입력한 가로 cm 비율에 맞춰 달라집니다.
- 블라인드 세로 배지는 `세로 260+10cm(하단 10cm여유)`처럼 계산 과정을 함께 보여줍니다.
- 가운데 구간에는 가로 여유 체크박스를 표시하지 않습니다.
- 단가와 견적 금액은 계산하지 않습니다.
